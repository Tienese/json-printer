JSON Format Guide:

Your JSON should follow this structure:

{
  "title": "Quiz Title",
  "description": "Quiz description",
  "questions": [
    {
      "type": "MC",
      "title": "Question 1",
      "prompt": "What is 2+2?",
      "points": 1.0,
      "answers": [
        {"text": "3", "correct": false},
        {"text": "4", "correct": true}
      ]
    }
  ]
}
Supported Types: MC (Multiple Choice), MA (Multiple Answer), MD (Multiple Dropdown), MT (Matching), TF (True/False)
